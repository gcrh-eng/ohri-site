# OHRI — Observatório Hidrológico do Riacho Itararé

**Programa técnico da GCRH Engenharia**

O OHRI (Observatório Hidrológico do Riacho Itararé) é uma iniciativa independente dedicada à produção, organização e difusão de informações hidrológicas urbanas, com foco na sub-bacia do riacho Itararé, em Teresina–PI.

Este repositório constitui uma plataforma técnica aberta para disponibilização de:

- estudos hidrológicos
- relatórios técnicos
- dados geoespaciais
- mapas temáticos
- metodologias e protocolos

---

## 🎯 Objetivo

Estruturar uma base consistente de conhecimento técnico sobre a dinâmica hidrológica urbana da sub-bacia do riacho Itararé, subsidiando:

- avaliação de risco de alagamentos  
- planejamento urbano e drenagem  
- estudos ambientais e de impacto  
- tomada de decisão por agentes públicos e privados

## 🗺️ Dados e Geoinformação

Os dados disponibilizados incluem:

- pontos de alagamento  
- camadas temáticas derivadas  

### Formatos disponíveis:
- Shapefile (.shp)
- GeoPackage (.gpkg)
- PDF (mapas)
- CSV (quando aplicável)

### Sistema de referência:
SIRGAS 2000 / UTM (zona a definir conforme dataset)

## 🔬 Metodologia

Os produtos do OHRI são desenvolvidos com base em:

- análise geoespacial (SIG)
- modelagem hidrológica (chuva-vazão)
- interpretação de uso do solo
- análise de eventos hidrológicos urbanos

As limitações e premissas de cada estudo são explicitadas nos relatórios e metadados.
